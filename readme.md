### :gear: Installation & Usage

Clone the project

```bash
  git clone https://github.com/aaafarrr/Bot-Shopee-Flash-Sale-Auto-Checkout.git
```

```bash
  cd Bot-Shopee-Flash-Sale-Auto-Checkout
```

```bash
  php run.php
```
